1.compare_wav_files.py
This script compares the list of WAV files in one folder with those in another folder.
It returns the list of WAV files that are missing from the other folder.
2.filepathstocsv.py
This script writes the paths of all WAV files (including those in subdirectories) to a CSV file.
3.fp&sr&len_tocsv.py
This script writes the paths,samplerate,length of all WAV files (including those in subdirectories) to a CSV file.
